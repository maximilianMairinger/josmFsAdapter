# Josm fs adapter

Josm adapter for FileSystem storage.

> Please note that Josm fs adapter is currently under development and not yet suited for production

## Installation

```shell
 $ npm i josm-fs-adapter
```

## Usage



```ts
import josmFsAdapter from "josm-fs-adapter"

josmFsAdapter()
```
